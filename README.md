A minimal Next.js app that demos the [`emoji-recsys`](https://github.com/zsiegel92/emoji-recsys) package — a client-side emoji recommendation engine.

Type text into the search box and get relevant emoji suggestions in real time, powered entirely in the browser.

## Getting started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to see it in action.
